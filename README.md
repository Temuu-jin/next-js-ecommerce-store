# Welcome to Yazu Sushi

The best sushi in Mayfair!

## Technologies used:

- Next.JS
- TypeScript
- SCSS
- HTML
- Docker
- Jest
- Playwright
- Ley
- Node.JS
- PostreSQL

## Setup

Install Dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install

```

Start Environment:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
